Agrega una nueva seccion al formulario de arquitectura hexagonal.

## Instrucciones

1. Recibir como argumento el nombre del concepto (ej: "Factory Pattern")
2. Asignar el siguiente numero disponible en sections.js
3. Crear el objeto con la estructura exacta:
   ```
   { num, title, formula, meaning, code: { php, python, javascript, typescript, java, csharp }, output }
   ```
4. Incluir codigo funcional y correcto en los 6 lenguajes
5. La formula debe usar notacion algebraica consistente con las demas
6. Verificar que sections.js no exceda 200 lineas. Si lo hace, dividir en archivos por seccion.
7. Probar que la app compila con `npm run build`
