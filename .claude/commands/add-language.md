Agrega un nuevo lenguaje de programacion al formulario.

## Instrucciones

1. Recibir como argumento el nombre del lenguaje (ej: "Go", "Rust")
2. Agregar la clave al array LANGS en sections.js
3. Agregar la etiqueta en LANG_LABELS
4. Agregar el codigo equivalente en CADA una de las 12 secciones
5. Agregar reglas de tokenizacion en highlight.jsx (objeto RULES)
6. Verificar que ningun archivo exceda 200 lineas
7. Probar con `npm run build`
